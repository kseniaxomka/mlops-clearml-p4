Обзор проекта
В этой практической работе рассматриваются два современных инструмента для отслеживания экспериментов машинного обучения: MLflow и ClearML. 

Проект демонстрирует:
1)Отслеживание экспериментов машинного обучения
2)Логирование параметров, метрик и артефактов
3)Управление моделями через Model Registry
4)Подбор гиперпараметров
5)Работу с артефактами и визуализацией

Выполнение экспериментов:
1)MLflow эксперименты: Запуск скриптов с MLflow логированием
2)ClearML эксперименты: Запуск скриптов с инициализацией Task
3)Мониторинг:
  -MLflow: http://127.0.0.1:8080
  -ClearML: https://app.clear.ml

Преимущества MLflow:
1)Простота интеграции с существующим кодом
2)Локальный запуск без облачной инфраструктуры
3)Модельный реестр для управления жизненным циклом моделей
4)Поддержка множества фреймворков

Преимущества ClearML:
1)Автоматическое отслеживание всего эксперимента
2)Веб-интерфейс с расширенными возможностями
3)Коллаборативные функции для командной работы
4)Оркестрация распределенных вычислений
